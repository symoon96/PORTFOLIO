CSS

  • - 같은 분류 안에서 다른 스타일은 버전을 붙여 스타일 추가
  • - 서브 페이지에 필요한 콘텐츠 스타일은 페이지에 해당하는 id나 클래스를 타고 들어가서 추가
  • - common.css: 공통 레이아웃에 사용하는 스타일
  • - components.css: 반복적으로 사용되는 객체 스타일
  • - fonts.css: 웹폰트 선언 스타일
  • - main.css: 메인페이지 스타일
  • - reset.css: 리셋 스타일
  • - sub.css: 서브 페이지 스타일
    서브 페이지에 들어가는 공통 레이아웃이나 각 페이지에 들어가는 콘텐츠 스타일
  • - utility.css: 기존 스타일에서 커스텀하는데 필요한 스타일
